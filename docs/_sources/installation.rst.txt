============
Installation
============

Install from GitHub
===================
Clone the repository by

.. code-block:: bash

    git clone https://github.com/kh10000/st2d.git


and

.. code-block:: bash

    cd st2d
    pip install .

MPI parallelization
===================
**st2d** supports MPI parallelization. In order for parallel calculations, additional installation of **mpi4py** is required.

.. code-block:: bash

    pip install mpi4py
